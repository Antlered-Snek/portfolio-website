Programming Projects

I started learning how to program in mid-2023
So far I've been able to program in Python, C++, Javascript, and GDScript

  • Matrix Calculator

    A calculator made in Python for calculating matrices

    I made this around the time I was learning matrices at school, and added features along the way
    as I learned new material and ways to use matrices in class

    It was made on my phone and published on the website where I learned most the basics in programming

  • My Little Painter

    A basic art program I made for an English group assignment

    The assignment was to make a Procedure Presentation about anything, so I chose this opportunity
    to further test my abilities of pushing Javascript and HTML Canvas to its limits

    My team kindly agreed and allowed me to do my thing, while they're in charge of making the presentation

  • Mechanized Devil Hunter

    Work In Progress || Demo Published

    A top down shooter I made using Javascript and HTML Canvas
    A basic game with not so basic enemies as this was the first time I had the chance to program enemy AI

    I really focused on programming unique AI for each enemy encountered in this game, and recently just started to program more complex enemies

    Making this game has been such an eye opener for me when it comes to working with videogame AI
    I plan on finishing this game before 2026

  • Battle Beats

    Work In Progress || Demo Published

    A "Fighting" Game I made originally for a friend's birthday

    It's the first game I've ever made using any medium, and for some reason I thought it would be
    a good idea to start off using Javascript and HTML Canvas basically starting from scratch with no training wheels

    Despite being labelled as a Fighting Game, it is very different from anything else in the genre
    Mostly because I didn't know what I was doing at the time, but also because I wanted to make
    every playable character feel like a Metroidvania Boss

    I plan on adding more characters, maps, and of course, actual sounds and sprites instead of simple placeholder cubes

  • Brainrot Instagram Circle

    A very simple joke website based on a trend in the past that my friends thought was stupid

    Its creation originally started as a joke during lunch when I told my friends that I could easily recreate this virus that has been plaging their Instagram reels

    I ended up making it because I thought it would be funny

  • Full Metal Days

    Prototype

    A game made in the open source Godot game engine
    The only game I've made so far that was made in a real game engine and not just me butchering Javascript

    This game will emphasize realism over everything, with smart AI that can strategize mid-combat
    It's an ambitious plan, it might not even be fun to play in end with how different it is from other games
    despite taking some surface-level inspiration from the Armored Core Franchise

    I plan to fully develop this game in the far future, so far I've gotten the broad strokes of the story written as well concept art, basic game mechanics, and smart AI groundwork

  • Birthday Website

    A website that my friend commissioned me to make for their other friend's birthday
    For privacy reasons, the version I have in my portfolio is a modified version without the customer's name and photos

    This was the first "true" website I've ever made
    Since the beginning, it was a challenge to format the website perfectly according to the customer's orders
    Even after I finished, I soon realized that this website is Broken when viewed from a mobile device

    However, I'm very proud of how it turned out. It's quite the eye-candy, packed with fun minigames and easter eggs
